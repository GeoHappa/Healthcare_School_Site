# Medical Coding Instructor

---

## Position Overview

**Title:** Medical Coding Specialist Program Instructor
**Department:** Academics
**Reports To:** Program Director
**Status:** Part-Time or Full-Time
**Salary Range:** $40-55/hour (part-time) / $50,000 - $65,000 annually (full-time)

---

## Position Summary

The Medical Coding Instructor is responsible for delivering comprehensive instruction in medical coding, including ICD-10-CM, ICD-10-PCS, CPT, and HCPCS Level II coding systems. This position prepares students for entry-level employment in medical coding and billing and for national certification examinations such as CPC (AAPC) or CCS (AHIMA).

## Certification Knowledge Requirements

The instructor must be proficient in ALL of the following coding systems:

### ICD-10-CM (Diagnosis Coding)
- [ ] Code structure and format
- [ ] Official Guidelines for Coding and Reporting
- [ ] Chapter-specific guidelines (all chapters)
- [ ] External cause codes
- [ ] Z codes (factors influencing health status)
- [ ] Manifestation codes
- [ ] Combination codes
- [ ] Sequencing rules

### ICD-10-PCS (Inpatient Procedure Coding)
- [ ] Code structure (7 characters)
- [ ] Medical and surgical section
- [ ] Root operations
- [ ] Body parts, approaches, devices
- [ ] Guidelines and conventions
- [ ] Obstetrics, placement, administration sections

### CPT (Current Procedural Terminology)
- [ ] Code structure and modifiers
- [ ] Evaluation and Management (E/M) coding
- [ ] 2021 E/M guideline changes
- [ ] Anesthesia section
- [ ] Surgery section (all subsections)
- [ ] Radiology, pathology, medicine
- [ ] Category II and III codes
- [ ] Unlisted codes and special reports

### HCPCS Level II
- [ ] Code structure
- [ ] DME codes
- [ ] Drug codes (J codes)
- [ ] Transportation codes
- [ ] Temporary codes

## Essential Duties and Responsibilities

### Classroom Instruction (70%)
- Deliver comprehensive lectures on all coding systems
- Teach medical terminology as it relates to coding
- Instruct in anatomy and physiology relevant to code selection
- Guide students through coding exercises and case studies
- Administer examinations (knowledge and practical coding)
- Provide individualized feedback on coding assignments
- Prepare students for certification examinations

### Curriculum Topics

**Medical Terminology & Anatomy (Foundation)**
- [ ] Body systems and structures
- [ ] Medical word building
- [ ] Abbreviations and symbols
- [ ] Disease processes
- [ ] Diagnostic procedures
- [ ] Treatment modalities

**Healthcare Reimbursement**
- [ ] Healthcare delivery systems
- [ ] Insurance types and payers
- [ ] Revenue cycle management
- [ ] Claims submission process
- [ ] Denials and appeals
- [ ] Compliance and fraud prevention

**Coding Systems**
- [ ] ICD-10-CM (diagnosis) - 80+ hours
- [ ] ICD-10-PCS (inpatient procedures) - 40+ hours
- [ ] CPT (outpatient procedures) - 80+ hours
- [ ] HCPCS Level II - 20+ hours
- [ ] Modifiers (all systems) - 20+ hours

**Coding Practice**
- [ ] Code selection exercises
- [ ] Medical record abstraction
- [ ] Real-world case studies
- [ ] Mock certification exams
- [ ] Speed and accuracy drills

### Student Support (20%)
- Provide tutoring and remediation
- Monitor student progress
- Identify struggling students early
- Assist with certification exam preparation
- Answer student questions on coding scenarios
- Connect concepts to real-world application

### Administrative Duties (10%)
- Maintain accurate attendance and grade records
- Update curriculum for annual code changes (ICD-10, CPT updates)
- Order current coding manuals and resources
- Attend faculty meetings
- Participate in curriculum review
- Maintain professional credentials

## Sample Interview Questions

### Credentials & Experience
1. What coding certifications do you hold? When did you obtain them?
2. Describe your medical coding experience. What settings have you worked in?
3. How do you maintain your certification? What CEUs have you completed recently?
4. Have you trained or mentored other coders?

### Technical Knowledge
5. Explain the structure of an ICD-10-CM code and how you would look up a diagnosis.
6. What are the seven characters in ICD-10-PCS and what does each represent?
7. Walk me through how you would code an E/M visit using the 2021 guidelines.
8. How do you determine medical necessity for a procedure?

### Teaching Ability
9. How would you explain the concept of "root operations" in ICD-10-PCS to a new student?
10. A student is struggling with modifier selection. How do you help them?
11. How do you keep students engaged during lengthy coding lectures?
12. How do you prepare students for certification exams?

### Practical Coding Test
13. Please code these 10 diagnoses from the documentation provided.
14. Code these 5 E/M encounters and justify your level selection.
15. Review this operative report and assign the appropriate CPT codes.

### Situational
16. The new ICD-10-CM update has significant changes. How do you update your curriculum?
17. A student insists their code selection is correct, but it's not. How do you handle it?
