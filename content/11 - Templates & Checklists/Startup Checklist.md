# Startup Checklist - Healthcare Training Center

## Overview

This master checklist tracks all major tasks required to launch a healthcare training center in California. Use this alongside the [[Master Timeline]] to ensure nothing is missed.

---

## Phase 1: Planning & Research (Months 1-3)

### Business Planning
- [ ] Complete market research
- [ ] Analyze competitors
- [ ] Define program offerings
- [ ] Create business plan
- [ ] Develop financial projections (3-year)
- [ ] Identify funding sources
- [ ] Establish budget

### Legal & Structure
- [ ] Choose business structure (LLC/Corp)
- [ ] File with CA Secretary of State
- [ ] Obtain EIN from IRS
- [ ] Register business locally
- [ ] Consult with education attorney
- [ ] Review insurance requirements

### Facility Research
- [ ] Define space requirements
- [ ] Research Folsom area locations
- [ ] Tour potential sites
- [ ] Get contractor estimates for build-out
- [ ] Verify zoning for educational use

---

## Phase 2: BPPE Application (Months 3-6)

### Application Preparation
- [ ] Download current BPPE application
- [ ] Attend BPPE application workshop
- [ ] Prepare ownership documents
- [ ] Prepare financial statements
- [ ] Develop curriculum documents
- [ ] Create enrollment agreement draft
- [ ] Create school catalog draft
- [ ] Create SPFS template
- [ ] Gather instructor credentials
- [ ] Secure medical advisor agreement

### Facility Secured
- [ ] Sign lease agreement
- [ ] Verify Certificate of Occupancy
- [ ] Confirm ADA accessibility
- [ ] Complete fire safety inspection
- [ ] Obtain zoning approval letter

### Application Submission
- [ ] Complete all application forms
- [ ] Compile all attachments
- [ ] Review for completeness
- [ ] Submit application + $5,000 fee
- [ ] Receive confirmation and file number

---

## Phase 3: Facility Setup (Months 4-8)

### Build-Out
- [ ] Obtain building permits
- [ ] Complete construction/renovation
- [ ] Install flooring
- [ ] Paint and finish
- [ ] Install lighting
- [ ] Plumbing work (lab sink)
- [ ] Electrical upgrades
- [ ] HVAC modifications
- [ ] Install signage

### Equipment Procurement
- [ ] Order clinical lab equipment
- [ ] Order computer equipment
- [ ] Order classroom furniture
- [ ] Order office furniture
- [ ] Install all equipment
- [ ] Test all systems
- [ ] Set up lab supplies inventory

### Technology Setup
- [ ] Set up network infrastructure
- [ ] Install computers
- [ ] Configure software:
  - [ ] Student Information System
  - [ ] EHR training software
  - [ ] Coding software
  - [ ] Learning Management System
  - [ ] Accounting software
- [ ] Set up phone system
- [ ] Set up email system
- [ ] Create website

---

## Phase 4: Program Development (Months 5-8)

### Curriculum Finalization
- [ ] Finalize MA curriculum
- [ ] Finalize Medical Coding curriculum
- [ ] Finalize Phlebotomy curriculum (if applicable)
- [ ] Finalize EKG curriculum (if applicable)
- [ ] Develop course materials
- [ ] Create competency checklists
- [ ] Order textbooks

### Phlebotomy Program (Additional)
- [ ] Submit CDPH program approval application
- [ ] Develop CDPH-compliant curriculum
- [ ] Identify clinical training sites for phlebotomy
- [ ] Receive CDPH approval

---

## Phase 5: Staffing (Months 6-9)

### Hiring
- [ ] Develop job descriptions
- [ ] Post positions
- [ ] Conduct interviews
- [ ] Hire Program Director
- [ ] Hire Lead MA Instructor
- [ ] Hire Coding Instructor
- [ ] Hire Admin/Admissions Coordinator
- [ ] Conduct background checks
- [ ] Verify credentials

### Onboarding
- [ ] Create employee handbook
- [ ] Develop training materials
- [ ] Conduct BPPE compliance training
- [ ] Conduct FERPA training
- [ ] Orient staff to facility
- [ ] Train on systems and software

---

## Phase 6: Clinical Partnerships (Months 4-9)

### Partnership Development
- [ ] Identify target clinical sites
- [ ] Develop partnership presentation
- [ ] Conduct outreach to hospitals
- [ ] Conduct outreach to clinics
- [ ] Conduct outreach to labs
- [ ] Present to potential partners
- [ ] Negotiate terms

### Formalize Agreements
- [ ] Draft affiliation agreement template
- [ ] Legal review of agreements
- [ ] Execute MA externship agreements (minimum 5)
- [ ] Execute phlebotomy clinical agreements (minimum 3)
- [ ] Document all agreements on file

### Advisory Board
- [ ] Identify advisory board candidates
- [ ] Invite members
- [ ] Schedule first meeting
- [ ] Establish meeting schedule

---

## Phase 7: Compliance & Policies (Months 7-10)

### Policy Development
- [ ] Attendance policy
- [ ] Grading policy
- [ ] Academic probation policy
- [ ] Student conduct policy
- [ ] Grievance procedure
- [ ] Refund policy (BPPE compliant)
- [ ] FERPA policy
- [ ] Drug-free school policy
- [ ] Title IX policy
- [ ] Emergency procedures

### Documents Finalization
- [ ] Finalize School Catalog
- [ ] Finalize Enrollment Agreement
- [ ] Finalize School Performance Fact Sheet
- [ ] Create student handbook
- [ ] Create faculty handbook

### Insurance
- [ ] General liability insurance
- [ ] Professional liability insurance
- [ ] Worker's compensation
- [ ] Property insurance (if needed)

---

## Phase 8: BPPE Approval (Months 9-12)

### Pre-Site Visit
- [ ] Respond to any BPPE information requests
- [ ] Ensure facility is complete
- [ ] Ensure all equipment installed
- [ ] Ensure staff hired and trained
- [ ] Organize all required documents
- [ ] Prepare for site visit

### Site Visit
- [ ] BPPE schedules site visit
- [ ] Conduct walk-through
- [ ] Answer inspector questions
- [ ] Provide requested documents
- [ ] Address any findings

### Approval
- [ ] Receive Approval to Operate letter
- [ ] Post approval certificate
- [ ] Update marketing materials with approval status

---

## Phase 9: Marketing Launch (Months 8-11)

### Brand Development
- [ ] Develop brand identity (logo, colors)
- [ ] Create style guide
- [ ] Design marketing materials
- [ ] Print brochures and flyers

### Digital Presence
- [ ] Launch website
- [ ] Set up Google Business Profile
- [ ] Create social media accounts
- [ ] Set up Google Analytics
- [ ] Install tracking pixels

### Marketing Campaigns
- [ ] Launch Google Ads
- [ ] Launch Facebook/Instagram ads
- [ ] Begin SEO efforts
- [ ] Plan community events
- [ ] Develop email marketing

### Community Outreach
- [ ] Connect with high school counselors
- [ ] Contact workforce development agencies
- [ ] Plan open house events
- [ ] Identify local event opportunities

---

## Phase 10: First Enrollment (Months 10-12)

### Enrollment Preparation
- [ ] Finalize tuition and fees
- [ ] Set up payment processing
- [ ] Train admissions staff
- [ ] Create enrollment workflow
- [ ] Prepare enrollment packets

### Student Recruitment
- [ ] Begin accepting inquiries
- [ ] Conduct campus tours
- [ ] Process applications
- [ ] Conduct enrollment appointments
- [ ] Collect deposits

### Pre-Start Preparation
- [ ] Verify student documentation
- [ ] Order student materials
- [ ] Create class schedules
- [ ] Assign externship sites
- [ ] Plan orientation

---

## Phase 11: Operations Launch (Month 12+)

### Orientation
- [ ] Conduct new student orientation
- [ ] Distribute materials
- [ ] Complete remaining paperwork
- [ ] Introduce staff and instructors

### Classes Begin
- [ ] First day of classes
- [ ] Monitor attendance
- [ ] Provide student support
- [ ] Address any issues promptly

### Ongoing Operations
- [ ] Weekly team meetings
- [ ] Monthly financial review
- [ ] Student progress monitoring
- [ ] Enrollment for next cohort
- [ ] Plan for accreditation

---

## Phase 12: Stabilization & Accreditation (Months 13-24+)

### Operations Refinement
- [ ] Gather student feedback
- [ ] Refine curriculum as needed
- [ ] Improve processes
- [ ] Expand enrollment

### Outcome Tracking
- [ ] Track completion rates
- [ ] Track certification pass rates
- [ ] Track job placement rates
- [ ] Document for SPFS

### Accreditation Planning
- [ ] Research ABHES requirements
- [ ] Begin self-study
- [ ] Compile documentation
- [ ] Submit accreditation application

---

## Quick Reference: Critical Path Items

These items have the longest lead times - start early!

1. **BPPE Application** - 6-12 month processing
2. **CDPH Phlebotomy Approval** - 2-4 months
3. **Clinical Partnership Agreements** - 3-6 months
4. **Facility Build-Out** - 2-4 months
5. **Accreditation** - 18-24 months after launch
