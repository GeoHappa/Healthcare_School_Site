# Resume Assistance

---

## Overview

Career Services provides free resume assistance to help students and graduates create professional resumes that effectively highlight their healthcare training and qualifications.

---

## Services Available

### Resume Development
- Create new professional resume
- Convert existing resume for healthcare field
- Format for ATS (Applicant Tracking Systems)
- Tailor for specific job applications

### Resume Review
- Critique of existing resume
- Suggestions for improvement
- Error checking and proofreading
- Format and layout recommendations

### Cover Letter Assistance
- Introduction to cover letter writing
- Templates for healthcare positions
- Customization for specific employers
- Review and feedback

### LinkedIn Profile Help
- Profile setup and optimization
- Professional summary writing
- Skills and endorsements
- Industry connections

### During Your Program
- Start building resume early
- Add clinical experiences as you go
- Include certifications as earned

### Before Externship
- Prepare resume for externship sites
- Many sites request student resumes
- Professional presentation matters

### Before Graduation
- Final resume polish
- Prepare for job search
- Career Services appointment

### After Graduation
- Update with final credential
- Tailor for job applications
- Ongoing support available

## Resume Sections for Healthcare

### Contact Information
- Full name
- Phone number (professional voicemail)
- Professional email address
- City, State (full address optional)
- LinkedIn URL (optional)

### Professional Summary
Brief statement highlighting:
- Career objective or summary
- Key qualifications
- Relevant skills
- Career focus

### Education
- [SCHOOL NAME], City, State
- Program completed and credential earned
- Graduation date (or expected)
- GPA (if 3.0 or higher)
- Relevant coursework (optional)

### Certifications/Licenses
- Certification name and acronym
- Certifying organization
- Date obtained
- License number (if applicable)

### Clinical Experience/Externship
- Site name and location
- Dates of clinical
- Hours completed
- Key responsibilities and skills used

### Skills
**Clinical Skills:**
- Venipuncture, injections, vital signs
- EKG administration
- Medical coding (ICD-10, CPT)
- EMR/EHR systems

**Administrative Skills:**
- Scheduling, patient intake
- Insurance verification
- Medical records

**Technical Skills:**
- Specific software proficiency
- Equipment operation

### Work Experience
- Previous employment (healthcare-related first)
- Transferable skills from other fields
- Volunteer experience

## Resume Tips for New Graduates

### Highlight Training
- Clinical hours completed
- Skills learned in lab
- Certification exam preparation
- Number of venipunctures (phlebotomy)

### Use Action Verbs
- Performed, administered, collected
- Documented, verified, processed
- Assisted, prepared, maintained
- Communicated, coordinated, trained

### Quantify When Possible
- "Performed 75+ venipunctures during clinical"
- "Processed 20+ patients daily"
- "Maintained 95% attendance"

### Emphasize Soft Skills
- Professional communication
- Attention to detail
- Team collaboration
- Patient care focus
- Time management

## Common Mistakes to Avoid

| Mistake | Solution |
|---------|----------|
| Typos and errors | Proofread multiple times, have someone review |
| Unprofessional email | Create professional email (firstname.lastname@) |
| Too long | Keep to 1 page for new graduates |
| Generic objective | Customize for each position |
| Missing clinical experience | Include externship details |
| No certifications section | Create dedicated section |
| Dense text | Use bullets and white space |

## Resume Formats

### Chronological (Recommended)
- Lists experience in reverse chronological order
- Best for those with relevant experience
- Most common and expected format

### Functional
- Emphasizes skills over dates
- Useful for career changers
- May raise questions about experience gaps

### Combination
- Highlights skills AND experience
- Good for those with transferable skills
- More complex format

## ATS Optimization

Many employers use Applicant Tracking Systems. To optimize:

### Do
- Use standard section headings
- Include keywords from job posting
- Use simple formatting
- Submit as .docx or PDF (as requested)
- Use standard fonts

### Don't
- Use tables or columns
- Include graphics or images
- Use headers/footers for key info
- Use unusual fonts
- Submit incompatible file formats
