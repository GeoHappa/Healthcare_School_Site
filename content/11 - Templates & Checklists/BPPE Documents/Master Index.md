## How to Use This Vault

### For BPPE Application
1. Start with [[BPPE Documents Index]] for application-specific guidance
2. Customize all documents with school-specific information
3. Replace all `[BRACKETED]` placeholders
4. Have documents reviewed by education attorney
5. Submit with BPPE application

### For Operations
1. Use policies as operational guides
2. Use forms for day-to-day operations
3. Keep documents updated annually
4. Train staff on all policies

### Placeholder Guide
| Placeholder | Replace With |
|-------------|--------------|
| `[SCHOOL NAME]` | Your legal school name |
| `[ADDRESS]` | Physical address |
| `[PHONE]` | Main phone number |
| `[EMAIL]` | Main email |
| `[DATE]` | Effective date |
| `[CODE]` | BPPE institution code |
| `$_______` | Actual tuition amounts |
| `[X]` | Specific numbers/weeks |
