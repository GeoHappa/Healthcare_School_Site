## How to Use These Templates

### Before Submitting to BPPE

1. **Replace all placeholders** - Search for `[SCHOOL NAME]`, `[DATE]`, `[ADDRESS]`, etc.
2. **Complete all blank fields** - Fill in tuition amounts, schedules, program details
3. **Review for accuracy** - Ensure all information matches your actual operations
4. **Legal review** - Have an education attorney review before submission
5. **Get BPPE guidance** - Attend BPPE workshop for current requirements

### Placeholder Guide

| Placeholder | Replace With |
|-------------|--------------|
| `[SCHOOL NAME]` | Your school's legal name |
| `[ADDRESS]` | Physical location |
| `[PHONE]` | Main phone number |
| `[EMAIL]` | Main contact email |
| `[DATE]` | Effective date |
| `[BPPE CODE]` | Institution code (after approval) |
| `$________` | Actual dollar amounts |

These documents must be updated annually:
