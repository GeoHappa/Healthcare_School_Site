# Assessment Methods

---

## Overview

[SCHOOL NAME] uses multiple assessment methods to evaluate student learning and ensure graduates are prepared for healthcare careers. This document describes the types of assessments used across all programs.

---

## Purpose of Assessment

Assessments serve to:
- Measure student mastery of content
- Identify areas needing additional support
- Ensure competency before clinical placement
- Prepare students for certification exams
- Document academic progress
- Meet regulatory requirements

---

## Types of Assessments

### Written Assessments

#### Quizzes
| Feature | Description |
|---------|-------------|
| Frequency | Weekly or per module |
| Length | 10-25 questions |
| Format | Multiple choice, true/false, fill-in |
| Weight | Typically 15-20% of grade |
| Purpose | Check understanding of recent material |

#### Module/Unit Exams
| Feature | Description |
|---------|-------------|
| Frequency | End of each module |
| Length | 25-50 questions |
| Format | Multiple choice, matching, short answer |
| Weight | Typically 30-40% of grade |
| Purpose | Comprehensive assessment of module content |

#### Comprehensive Final Exam
| Feature | Description |
|---------|-------------|
| Frequency | End of program/course |
| Length | 50-100+ questions |
| Format | Similar to certification exam format |
| Weight | Typically 15-25% of grade |
| Purpose | Assess overall program knowledge |

---

### Practical Skills Assessments

#### Skills Checkoffs
| Feature | Description |
|---------|-------------|
| Frequency | After skills instruction |
| Format | Live demonstration with checklist |
| Passing | Must complete all critical steps |
| Attempts | Up to 3 attempts (see [[Retake Policy]]) |
| Weight | Part of practical/lab grade |

**Evaluated On:**
- Proper technique
- Safety procedures
- Patient communication
- Time management
- Accuracy

#### Lab Practicals
| Feature | Description |
|---------|-------------|
| Frequency | Mid-term and/or final |
| Format | Multiple skills in timed setting |
| Passing | 70% minimum |
| Purpose | Demonstrate competency in skills |

---

### Clinical Assessments

#### Externship Evaluations
| Feature | Description |
|---------|-------------|
| Frequency | Weekly and final |
| Evaluator | Clinical site supervisor |
| Format | Standardized evaluation form |
| Areas | Clinical skills, professionalism, attendance |

#### Clinical Competency Logs
| Feature | Description |
|---------|-------------|
| Format | Documentation of procedures performed |
| Requirement | Minimum numbers for specific procedures |
| Validation | Supervisor signature required |

**Example (Phlebotomy):**
- 50 successful venipunctures
- 10 successful skin punctures

---

### Other Assessments

#### Assignments/Homework
| Feature | Description |
|---------|-------------|
| Frequency | Weekly |
| Format | Worksheets, case studies, projects |
| Weight | Typically 10-15% of grade |
| Purpose | Apply and reinforce learning |

#### Class Participation
| Feature | Description |
|---------|-------------|
| Components | Attendance, engagement, contribution |
| Weight | Typically 5-10% of grade |
| Purpose | Encourage active learning |

#### Research Projects
| Feature | Description |
|---------|-------------|
| Frequency | 1-2 per program |
| Format | Written report and/or presentation |
| Topics | Healthcare-related research topics |
| Purpose | Develop research and communication skills |

## Grading Standards

### Passing Scores

| Assessment Type | Minimum Passing |
|-----------------|-----------------|
| Written exams | 70% |
| Skills checkoffs | 100% of critical steps |
| Lab practicals | 70% |
| Overall course grade | 70% (C) |

### Grading Scale

| Grade | Percentage | GPA Points |
|-------|------------|------------|
| A | 90-100% | 4.0 |
| B | 80-89% | 3.0 |
| C | 70-79% | 2.0 |
| D | 60-69% | 1.0 |
| F | Below 60% | 0.0 |

See [[Grading Scale]] for complete details.

## Testing Conditions

### Written Exams
- Closed book (unless specified)
- No electronic devices
- Time limits enforced
- Proctored environment

### Skills Assessments
- Instructor or qualified evaluator observes
- Checklist used for consistency
- All safety procedures must be followed
- Time limits may apply

### Certification Exam Simulation
- Final exams simulate certification format
- Timed conditions
- Computer-based (when possible)
- No reference materials

---

## Accommodations

Students with documented disabilities may receive:
- Extended time
- Separate testing location
- Modified format
- Assistive technology

See [[Disability Services (ADA)]] for request procedures.

---

## Academic Integrity

All assessments are subject to [[Academic Integrity Policy]].

**Prohibited:**
- Copying from others
- Using unauthorized materials
- Sharing exam content
- Falsifying clinical documentation

---

## Retakes and Make-Ups

### Exam Retakes
- Available for failing scores (below 70%)
- Maximum attempts per [[Retake Policy]]
- Remediation required before retake

### Skills Retakes
- Required if critical steps missed
- Additional practice required
- Maximum 3 attempts

### Make-Up Exams
- Available for documented excused absences
- See [[Make-Up Policy]] for procedures
- May be different version

---

## Feedback

### Receiving Feedback
Students receive:
- Written exam scores within 48 hours
- Skills feedback immediately after checkoff
- Detailed feedback on request
- Progress reports at evaluation points

### Using Feedback
- Review incorrect answers
- Identify patterns of errors
- Seek tutoring for weak areas
- Apply feedback to future work

---

## Appealing Grades

If you believe an assessment was graded unfairly:
1. Discuss with instructor first
2. Review grading criteria
3. File formal appeal if needed

See [[Grade Appeals]] for complete process.
