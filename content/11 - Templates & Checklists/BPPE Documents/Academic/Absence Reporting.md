# Absence Reporting

---

## Notification Requirements

**You MUST notify the school BEFORE class begins** when you will be absent.

---

## How to Report an Absence

### Phone
- Call: [PHONE NUMBER]
- Available: 7:00 AM - 10:00 PM daily
- Leave voicemail if after hours

### Email
- Send to: [EMAIL ADDRESS]
- Include: Name, program, date(s), reason
- Send before class start time

### Text (if available)
- Text: [PHONE NUMBER]
- Include: Name and date(s)

---

## What to Include

When reporting an absence, provide:
1. Your full name
2. Program name
3. Date(s) you will be absent
4. Reason for absence
5. Expected return date
6. Contact number for questions

---

## Excused vs. Unexcused Absences

### Excused Absences
*Documentation required within 3 business days*

| Reason | Documentation Required |
|--------|----------------------|
| Illness | Doctor's note |
| Medical appointment | Appointment verification |
| Medical emergency | Hospital/doctor documentation |
| Death in family | Obituary or funeral notice |
| Court appearance | Court summons or verification |
| Military duty | Orders or official letter |
| Jury duty | Jury summons |

### Unexcused Absences

| Reason | Status |
|--------|--------|
| No call/no show | Unexcused |
| Personal appointments | Unexcused |
| Transportation issues | Unexcused |
| Work conflicts | Unexcused |
| Oversleeping | Unexcused |
| Childcare issues | Unexcused |
| No documentation provided | Unexcused |

---

## Documentation Submission

**Timeline:** Within 3 business days of return

**Submit to:** Front desk or Student Services

**Acceptable formats:**
- Original document
- Photocopy
- Scanned/emailed copy

---

## Failure to Report (No Call/No Show)

### Consequences

| Occurrence | Action |
|------------|--------|
| 1st No Call/No Show | Verbal warning, contact attempt |
| 2nd No Call/No Show | Written warning |
| 3rd No Call/No Show | Attendance probation |
| 3 Consecutive Days | Assumed withdrawn (unless contacted) |

---

## Extended Absences

If you will be absent for **3 or more consecutive days:**
- Contact Student Services
- Discuss options (make-up, LOA)
- Provide medical documentation if applicable
- See [[Leave of Absence]] for extended time off

---

## Returning from Absence

When you return:
1. Check in with instructor
2. Submit documentation (if applicable)
3. Get missed assignments/notes
4. Schedule make-up time (if eligible)
5. Review what was covered
