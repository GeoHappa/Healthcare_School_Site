# Grading Scale

---

## Letter Grades

| Letter | Percentage | Grade Points | Description |
|--------|------------|--------------|-------------|
| A | 90-100% | 4.0 | Excellent |
| B | 80-89% | 3.0 | Good |
| C | 70-79% | 2.0 | Satisfactory |
| D | 60-69% | 1.0 | Below Standards |
| F | 0-59% | 0.0 | Failing |

---

## Minimum Passing Grade

**70% (C) is required to pass all courses**

- Grades below 70% require course repeat
- All courses must be passed to progress
- Failed courses count toward maximum time frame

---

## Administrative Grades

| Grade | Meaning | GPA Impact |
|-------|---------|------------|
| I | Incomplete | No (until resolved) |
| W | Withdrawal | No |
| TC | Transfer Credit | No |
| P | Pass (Externship) | No |
| NP | No Pass (Externship) | No |

### Incomplete (I)
- Granted for extenuating circumstances
- Must be resolved within 2 weeks
- Converts to F if not completed
- See [[Incomplete Grade Policy]]

### Withdrawal (W)
- Student officially withdrew
- No grade point calculation
- Counts as attempted hours

### Transfer Credit (TC)
- Credits from another institution
- No grade point calculation
- Counts as completed hours

### Pass/No Pass (P/NP)
- Used for externship only
- P = equivalent to C or better
- NP = below passing standards
- See [[Externship Grading]]

## Grade Point Average (GPA)

### Calculation

```
GPA = Total Quality Points รท Total Credit Hours Attempted
```

**Quality Points = Grade Points ร Credit Hours**

### Example Calculation

| Course | Grade | Points | Hours | Quality Points |
|--------|-------|--------|-------|----------------|
| Anatomy | A | 4.0 | 3 | 12.0 |
| Medical Term | B | 3.0 | 2 | 6.0 |
| Clinical Proc | C | 2.0 | 4 | 8.0 |
| **Total** | | | **9** | **26.0** |

**GPA = 26.0 รท 9 = 2.89**

## Grade Reports

### Mid-Term Progress Report
- Issued at course midpoint
- Shows current standing
- Identifies at-risk students

### Final Grades
- Posted within 5 business days of course end
- Available through student portal
- Official transcript on request
