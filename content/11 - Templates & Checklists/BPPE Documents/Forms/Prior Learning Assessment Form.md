## Assessment Process

### How It Works

1. **Application Review:** Complete application and documentation reviewed
2. **Eligibility Determination:** Program Director determines if assessment warranted
3. **Assessment Method:** May include:
   - Documentation review
   - Skills demonstration
   - Written examination
   - Competency testing
4. **Credit Determination:** Hours awarded based on demonstrated competency
5. **Notification:** Student informed of results within [X] business days

### Possible Outcomes

| Result | Action |
|--------|--------|
| Full credit awarded | Tuition adjusted, courses waived |
| Partial credit awarded | Some courses waived |
| No credit awarded | Student completes all courses |
| Additional assessment needed | Schedule testing |
