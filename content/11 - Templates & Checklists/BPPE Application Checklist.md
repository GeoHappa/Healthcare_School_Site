# BPPE Application Checklist

## Overview

Use this checklist to ensure your BPPE Application for Approval to Operate is complete before submission. Incomplete applications cause significant delays.

---

## Pre-Application Preparation

### Research & Planning
- [ ] Review California Education Code sections 94800-94950
- [ ] Read BPPE regulations (5 CCR Division 7.5)
- [ ] Attend BPPE application workshop (free, monthly)
- [ ] Download current application forms from bppe.ca.gov
- [ ] Consult with education attorney
- [ ] Review sample applications (if available)

### Business Formation
- [ ] Form legal entity (LLC or Corporation)
- [ ] Obtain EIN from IRS
- [ ] Register with California Secretary of State
- [ ] File Statement of Information
- [ ] File Fictitious Business Name (if applicable)
- [ ] Obtain local business license

---

## Application Documents Checklist

### Section A: Ownership Information

- [ ] **A1: Application Cover Sheet**
  - School name
  - Physical address
  - Mailing address
  - Contact information

- [ ] **A2: Ownership Disclosure**
  - List all owners with 10%+ interest
  - Include: Name, address, % ownership
  - For corporations: All officers, directors, shareholders
  - Background questionnaire for each owner

- [ ] **A3: Organizational Documents**
  - [ ] Articles of Incorporation/Organization
  - [ ] Bylaws or Operating Agreement
  - [ ] Statement of Information (filed with CA SOS)
  - [ ] Organizational chart

### Section B: Financial Information

- [ ] **B1: Financial Statements**
  - [ ] Audited financial statements (preferred)
  - [ ] OR CPA-prepared financial statements
  - [ ] Balance sheet
  - [ ] Income statement
  - [ ] Cash flow statement
  - [ ] Must show $50,000+ net worth OR post surety bond

- [ ] **B2: Evidence of Financial Stability**
  - [ ] Bank statements (3-6 months)
  - [ ] Line of credit documentation
  - [ ] Investor commitment letters (if applicable)

- [ ] **B3: Pro Forma Budget**
  - [ ] Projected revenue (Year 1-2)
  - [ ] Projected expenses (Year 1-2)
  - [ ] Assumptions documented

- [ ] **B4: Surety Bond (if required)**
  - Required if net worth < $50,000
  - Amount based on projected annual revenue

### Section C: Facility Information

- [ ] **C1: Facility Documentation**
  - [ ] Signed lease agreement OR deed
  - [ ] Lease term must cover approval period
  - [ ] Landlord contact information

- [ ] **C2: Floor Plan**
  - [ ] Scale drawing of facility
  - [ ] Room designations and square footage
  - [ ] Emergency exits marked

- [ ] **C3: Compliance Documentation**
  - [ ] Certificate of Occupancy (for educational use)
  - [ ] Fire safety inspection approval
  - [ ] ADA accessibility compliance
  - [ ] Zoning verification letter
  - [ ] Health permits (if required)

- [ ] **C4: Equipment List**
  - [ ] List of equipment by room
  - [ ] Evidence of ownership or lease

### Section D: Program Information

- [ ] **D1: Educational Objectives**
  - [ ] Mission statement
  - [ ] Institutional objectives
  - [ ] Program-specific objectives

- [ ] **D2: Program Approvals (for each program)**
  - [ ] Program name and credential awarded
  - [ ] Program description
  - [ ] Occupational outcomes
  - [ ] Clock hours or credit hours
  - [ ] Program length (weeks)

- [ ] **D3: Curriculum (for each program)**
  - [ ] Course titles and descriptions
  - [ ] Course clock hours
  - [ ] Learning objectives for each course
  - [ ] Methods of instruction
  - [ ] Assessment methods

- [ ] **D4: Externship/Clinical Information**
  - [ ] Externship requirements
  - [ ] Clinical site list
  - [ ] Affiliation agreements (draft acceptable)
  - [ ] Supervision plan

### Section E: Faculty and Staff

- [ ] **E1: Administrative Staff**
  - [ ] Organizational chart
  - [ ] Resumes for key personnel:
    - [ ] Owner/CEO
    - [ ] School Director
    - [ ] Program Director
    - [ ] Chief Financial Officer (if applicable)
  - [ ] Job descriptions

- [ ] **E2: Instructional Staff (for each instructor)**
  - [ ] Resume/CV
  - [ ] Copies of degrees/credentials
  - [ ] Verification of qualifications per Education Code 94310
  - [ ] Background check attestation

- [ ] **E3: Medical Advisor (for healthcare programs)**
  - [ ] Name and credentials
  - [ ] Active California license verification
  - [ ] Agreement letter to serve as advisor

### Section F: Student Information

- [ ] **F1: Enrollment Agreement (Draft)**
  - [ ] Per Education Code 94911 requirements
  - [ ] Include all required disclosures
  - [ ] Clear tuition and fees breakdown
  - [ ] Refund policy (pro-rata compliant)

- [ ] **F2: School Catalog (Draft)**
  Per Education Code 94909, must include:
  - [ ] School information and approval status
  - [ ] Mission and objectives
  - [ ] Program descriptions
  - [ ] Admission requirements
  - [ ] Academic calendar
  - [ ] Graduation requirements
  - [ ] Tuition and fees
  - [ ] Refund policy
  - [ ] Academic policies
  - [ ] Student services
  - [ ] Faculty qualifications
  - [ ] Grievance procedure

- [ ] **F3: School Performance Fact Sheet (Draft)**
  Per Education Code 94910, must include:
  - [ ] Completion rates
  - [ ] Placement rates
  - [ ] License exam pass rates
  - [ ] Salary/wage information
  - [ ] Loan default rates (if applicable)
  - [ ] (For new schools: May use "N/A" for first year)

- [ ] **F4: Student Policies**
  - [ ] Attendance policy
  - [ ] Grading policy
  - [ ] Student conduct policy
  - [ ] Grievance procedure
  - [ ] FERPA policy
  - [ ] Drug-free school policy

### Section G: Additional Requirements

- [ ] **G1: Application Fee**
  - [ ] $5,000 check or electronic payment
  - [ ] Payable to: Bureau for Private Postsecondary Education

- [ ] **G2: STRF Information**
  - [ ] STRF assessment rate acknowledgment
  - [ ] STRF disclosure language in enrollment agreement

- [ ] **G3: Advertising Samples (if any)**
  - [ ] Copies of all planned advertisements
  - [ ] Website screenshots
  - [ ] Brochures

## Post-Submission Process

### After Submitting Application
- [ ] Receive confirmation and file number
- [ ] Save all correspondence
- [ ] Prepare for information requests

### Responding to Deficiency Notices
- [ ] Respond within 30 days
- [ ] Address each item specifically
- [ ] Keep copies of everything submitted

### Site Visit Preparation
- [ ] Facility complete and operational
- [ ] All equipment in place
- [ ] Staff available for interviews
- [ ] Student files organized
- [ ] Policies posted as required

### After Approval
- [ ] Display approval certificate prominently
- [ ] Begin marketing and enrollment
- [ ] Submit Annual Report on schedule
- [ ] Maintain ongoing compliance

---

## Common Mistakes to Avoid

1. **Incomplete financial documentation** - CPA-prepared minimum
2. **Vague curriculum** - Include specific learning objectives
3. **Missing instructor credentials** - Verify ALL qualifications
4. **Non-compliant refund policy** - Must be pro-rata
5. **Facility not ready** - Have lease signed before applying
6. **Missing disclosures** - Follow enrollment agreement requirements exactly
7. **Unsigned documents** - Check all signature lines
