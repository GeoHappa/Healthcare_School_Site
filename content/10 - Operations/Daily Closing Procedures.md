# Daily Closing Procedures

> **Purpose:** Standardized checklist for closing the facility each day.
> **Responsible:** Last staff member to leave (typically Administrator or Instructor)
> **Time:** After last class/student departure

---

## End-of-Day Academic Tasks

### Attendance
- [ ] All attendance recorded in system
- [ ] Absences documented with reason (if known)
- [ ] Tardies noted
- [ ] Attendance reports saved

### Student Records
- [ ] Any student incidents documented
- [ ] Student files returned to secure storage
- [ ] Confidential documents secured
- [ ] FERPA compliance verified

### Instructor Tasks
- [ ] Daily teaching log completed
- [ ] Equipment properly stored
- [ ] Lab supplies inventoried if low
- [ ] Issues/concerns communicated to Program Director

---

## Facility Shutdown

### Classrooms
- [ ] Computers shut down (or logged off)
- [ ] Projector/AV equipment off
- [ ] Whiteboard erased
- [ ] Chairs pushed in/arranged
- [ ] Windows closed and locked
- [ ] Lights off

### Clinical Skills Lab
- [ ] All equipment powered off (except refrigerated items)
- [ ] Sharps containers checked - schedule pickup if 3/4 full
- [ ] Biohazard materials properly disposed
- [ ] Surfaces wiped down
- [ ] Supplies secured
- [ ] Lights off

### Computer Lab
- [ ] All computers shut down properly
- [ ] Printer off
- [ ] Confidential printouts removed from printer area
- [ ] Lights off

### Break Room/Student Areas
- [ ] Refrigerator checked (dispose of old food on Fridays)
- [ ] Coffee maker/appliances off and unplugged
- [ ] Tables wiped
- [ ] Trash emptied
- [ ] Lights off

### Restrooms
- [ ] Quick visual inspection
- [ ] Note any maintenance needs
- [ ] Lights off

---

## Administrative Closing

### Front Desk
- [ ] Computer shut down
- [ ] Cash/payments secured (if applicable)
- [ ] Visitor log filed
- [ ] Phone forwarded to voicemail
- [ ] Desk organized
- [ ] Confidential documents secured

### Office Areas
- [ ] Computers shut down or locked
- [ ] Sensitive documents filed/locked
- [ ] Office doors locked (if applicable)
- [ ] Lights off

### Filing/Records
- [ ] File cabinets locked
- [ ] Student records secured
- [ ] Financial documents secured

---

## Security Checklist

### Interior Security
- [ ] All interior doors locked (offices, storage)
- [ ] No students or visitors remaining
- [ ] All staff accounted for leaving
- [ ] Safe/cash drawer secured (if applicable)
- [ ] Confidential materials secured

### Exterior Security
- [ ] All windows closed and locked
- [ ] Back/side doors secured
- [ ] Parking lot clear of school vehicles
- [ ] Exterior lights set to auto/timer
- [ ] Main entrance double-locked

### System Security
- [ ] HVAC set to unoccupied/night mode
- [ ] Security system armed
- [ ] Motion sensors active
- [ ] Security cameras recording

---

## Final Walkthrough

### Visual Inspection Route
1. Start at front entrance
2. Walk through reception area
3. Check all classrooms
4. Inspect labs
5. Check restrooms
6. Walk hallways
7. Check break room
8. Verify back exit secured
9. Return to front entrance

### Look For
- [ ] All lights off
- [ ] No running water
- [ ] No unusual sounds
- [ ] No smoke or burning smells
- [ ] No unauthorized persons
- [ ] All doors secured

---

## Documentation

### Closing Log Entry
Record in Operations Log:
- Date and time closed
- Staff member closing
- Any issues noted
- Maintenance needs
- Items to address tomorrow

### Example Entry:
```
Date: [DATE]
Time Closed: 9:30 PM
Closed By: [Name]
Attendance Complete: Yes
Issues: [None / Describe issue]
Maintenance Needed: [List items]
Tomorrow Notes: [Any prep needed]
```

---

## Emergency Before Leaving

### If You Discover:
- **Water leak:** Shut off water main if possible, call Facility Manager
- **Electrical issue:** Do not touch, call Facility Manager
- **Break-in evidence:** Do NOT enter, call 911, then Director
- **Fire/smoke:** Evacuate, call 911, do NOT re-enter
- **Medical emergency:** Call 911, provide first aid if trained
