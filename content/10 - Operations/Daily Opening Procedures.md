# Daily Opening Procedures

> **Purpose:** Standardized checklist for opening the facility each day.
> **Responsible:** First staff member to arrive (typically Front Desk or Administrator)
> **Time:** 30-45 minutes before first class

---

## Pre-Opening Checklist (30 min before opening)

### Facility Access
- [ ] Unlock main entrance
- [ ] Disarm security system (code: ________)
- [ ] Turn on exterior lights (if dark)
- [ ] Check for any overnight issues (water leaks, damage, etc.)

### Climate Control
- [ ] Set HVAC to occupied mode
- [ ] Verify comfortable temperature (68-72Â°F)
- [ ] Check classroom temperatures

### Lighting
- [ ] Turn on reception area lights
- [ ] Turn on hallway lights
- [ ] Turn on classroom lights (as needed)
- [ ] Turn on restroom lights
- [ ] Check that all lights are functioning

### Front Desk Setup
- [ ] Turn on front desk computer
- [ ] Log into student information system
- [ ] Open attendance tracking system
- [ ] Check voicemail for overnight messages
- [ ] Review today's schedule
- [ ] Print daily attendance roster
- [ ] Prepare visitor sign-in sheet

### Common Areas
- [ ] Quick visual inspection of reception area
- [ ] Verify chairs/furniture arranged properly
- [ ] Check that magazines/brochures are stocked
- [ ] Ensure water cooler is filled (if applicable)
- [ ] Verify restrooms are clean and stocked

### Safety Check
- [ ] Verify all exit signs illuminated
- [ ] Check fire extinguisher locations (visual only)
- [ ] Ensure emergency exits are clear
- [ ] First aid kit accessible

---

## Classroom Preparation

### Main Classroom
- [ ] Lights on
- [ ] Computer/projector on and tested
- [ ] Whiteboard clean
- [ ] Markers available
- [ ] Chairs arranged
- [ ] Temperature comfortable
- [ ] Trash emptied from prior day

### Clinical Skills Lab
- [ ] Lights on
- [ ] Equipment powered on (as needed)
- [ ] Supplies stocked at stations
- [ ] Sharps containers checked (not overfilled)
- [ ] Hand sanitizer dispensers filled
- [ ] Sink areas clean

### Computer Lab
- [ ] All computers powered on
- [ ] Network connectivity verified
- [ ] Printer paper stocked
- [ ] Chairs arranged

---

## Communication Tasks

### Check Messages
- [ ] Review overnight voicemails
- [ ] Check general email inbox
- [ ] Review any text/SMS messages
- [ ] Note urgent items for immediate action

### Staff Communication
- [ ] Review staff schedule for today
- [ ] Note any instructor absences/substitutions
- [ ] Prepare for any scheduled meetings
- [ ] Post daily announcements (if any)

---

## Documentation

### Morning Log Entry
Record in Operations Log:
- Date and time opened
- Staff member opening
- Any issues noted
- Weather/road conditions (if notable)
- Any maintenance needs identified

### Example Entry:
```
Date: [DATE]
Time Opened: 7:30 AM
Opened By: [Name]
Issues: None / [Describe issue]
Notes: [Any relevant notes]
```

## Special Circumstances

### First Day of New Cohort
- [ ] Welcome signage displayed
- [ ] Enrollment packets ready
- [ ] Name tags prepared
- [ ] Classroom setup for orientation

### Testing Day
- [ ] Testing room prepared
- [ ] Test materials secured and ready
- [ ] "Testing in Progress" signs ready
- [ ] Quiet environment verified

### Site Visit Day (BPPE/Clinical Partner)
- [ ] All required documents accessible
- [ ] Facility extra-clean
- [ ] Staff notified
- [ ] Meeting room prepared
