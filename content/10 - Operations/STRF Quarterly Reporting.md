# STRF Quarterly Reporting Procedures

> **Purpose:** Step-by-step guide for completing quarterly Student Tuition Recovery Fund (STRF) reports to BPPE.
> **Responsible:** Finance Administrator / Director
> **Deadline:** 30 days after quarter end

---

## Quarterly Deadlines

| Quarter | Period | Report Due |
|---------|--------|------------|
| Q1 | January - March | **April 30** |
| Q2 | April - June | **July 31** |
| Q3 | July - September | **October 31** |
| Q4 | October - December | **January 31** |

---

## Current STRF Rate

**Effective April 1, 2024: $0.00 per $1,000**

Even when the rate is $0.00, quarterly reports must still be submitted.

---

## Pre-Reporting Checklist

### Gather Student Data (Week 3 of quarter-end month)
- [ ] Run enrollment report for quarter
- [ ] Identify all students who:
  - [ ] Were California residents at enrollment
  - [ ] Paid institutional charges during quarter
  - [ ] Did NOT have 100% third-party payment
- [ ] Calculate total institutional charges per student
- [ ] Verify STRF assessment amounts (if rate > $0)

### Required Information per Student
| Data Point | Source |
|------------|--------|
| Student full name | Enrollment agreement |
| Student SSN/TIN | Enrollment agreement |
| Date of enrollment | Student file |
| Program enrolled | Student file |
| Total institutional charges | Billing records |
| STRF assessment | Billing records |
| California residency | Enrollment agreement |

---

## Step-by-Step Reporting Process

### Step 1: Access BPPE Portal
1. Go to **connect.bppe.ca.gov**
2. Log in with institution credentials
3. Navigate to STRF Reporting section
4. Select correct quarter

### Step 2: Enter Institution Data
- Institution name
- Institution code (from approval letter)
- Report quarter
- Contact person information

### Step 3: Enter Student Summary Data

#### When Rate = $0.00
| Field | Entry |
|-------|-------|
| Number of eligible students | [Count] |
| Total institutional charges | [Sum] |
| STRF assessments collected | $0.00 |
| STRF assessments due | $0.00 |

#### When Rate > $0.00
| Field | Entry |
|-------|-------|
| Number of eligible students | [Count] |
| Total institutional charges | [Sum] |
| STRF assessments collected | [Total collected] |
| STRF assessments due | [Total due] |
| Remittance amount | [Check amount] |

### Step 4: Verify and Submit
- [ ] Review all data for accuracy
- [ ] Verify math calculations
- [ ] Submit report electronically
- [ ] Save confirmation number
- [ ] Print/save confirmation page

### Step 5: Payment (if rate > $0)
- [ ] Write check payable to "BPPE"
- [ ] Include institution code on check
- [ ] Include quarter reference on check
- [ ] Mail to:
  ```
  Bureau for Private Postsecondary Education
  P.O. Box 980818
  West Sacramento, CA 95798-0818
  ```

---

## Record Retention

### Keep on File for 5 Years
- [ ] Copy of submitted report
- [ ] Confirmation page
- [ ] Student roster for quarter
- [ ] Individual student STRF records
- [ ] Copy of remittance check (if applicable)
- [ ] Bank statement showing cleared check (if applicable)

### Student-Level Records
For each student assessed (or who would be assessed):
- Full name
- SSN/TIN
- Enrollment date
- Program
- Total charges paid
- STRF amount
- CA residency documentation

---

## Exemption Documentation

### Students NOT Subject to STRF
Document exemption reason for:
- Non-California residents (at enrollment)
- 100% third-party paid students
- Certain scholarship recipients

### Required Documentation
- [ ] Signed statement of non-residency, OR
- [ ] Third-party payment documentation, OR
- [ ] Scholarship documentation

---

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| Can't log in | Reset password or call BPPE |
| Student count discrepancy | Verify enrollment dates within quarter |
| Missing SSN | Contact student; cannot report without SSN/TIN |
| Late report | Submit ASAP; may incur citation |
| Math error caught post-submission | Contact BPPE for amendment process |

### Contact BPPE STRF Unit
- Phone: (916) 574-8900 (Option 5 or 6)
- Email: bppe.strf@dca.ca.gov

---

## Calendar Reminders

Set the following reminders:

| Reminder | When | Action |
|----------|------|--------|
| Start data gathering | Week 3 of last month of quarter | Pull reports |
| Data verification | Week 4 of last month | Verify accuracy |
| Submit report | By deadline | Complete submission |
| Confirm payment cleared | 2 weeks after deadline | Check bank statement |

---

## Quality Control

### Before Each Submission
- [ ] Double-check quarter dates
- [ ] Verify student count matches records
- [ ] Confirm institutional charges total
- [ ] Verify STRF calculation (if applicable)
- [ ] Check for prior quarter corrections needed

### Annual Reconciliation (January)
- [ ] Reconcile full year STRF data
- [ ] Match to annual report figures
- [ ] Identify any discrepancies
- [ ] Prepare corrections if needed
