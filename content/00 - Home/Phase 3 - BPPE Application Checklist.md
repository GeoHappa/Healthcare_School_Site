## Common Deficiencies to Avoid

| Issue | How to Avoid |
|-------|--------------|
| Missing signatures | Check every form |
| Incomplete ownership disclosure | List ALL owners 25%+ |
| Unqualified director/faculty | Verify credentials before listing |
| Refund policy errors | Have attorney review |
| Missing clinical agreements | Secure before submission |
| Incomplete curriculum | Detail all hours and topics |
| Financial insufficiency | Document adequate funding |

**After Submission:**
- Wait for BPPE acknowledgment
