# BPPE Application Elements - Form 94886

> **Complete Reference:** Every element required for BPPE Approval to Operate application (Non-Accredited Pathway). Use this document to prepare and verify your application before submission.

---

## Overview

The Application for Approval to Operate for an Institution Not Accredited (Form 94886, revision 8/24) requires 25 sections of documentation. Application fee: **$5,000**. Current processing time: **12-24+ months**.

**Two Pathways Exist:**
| Pathway | Form | Fee | For Whom |
|---------|------|-----|----------|
| Non-Accredited | 94886 | $5,000 | New schools without accreditation |
| Accredited | 94890 | $750 | Schools with USDE-recognized accreditation |

### Section 3: Owner(s) (5 CCR §71130)

#### 3.1 Ownership Disclosure
*Required for each person owning/controlling 25%+ or exercising substantial control*

| Element | Required |
|---------|----------|
| Name | Yes |
| Title | Yes |
| Physical Home Address | Yes |
| City, State, Zip | Yes |
| Federal Employer ID Number | If partnership |
| Social Security Number | If sole owner |
| Telephone Number | Yes |
| Email Address | Yes |
| Percentage of Ownership | Yes |
| Nature of Interest | Yes |

**"Substantial control" defined:** Authority to direct or cause the institution to expend **$5,000+** in any calendar year.

#### 3.2 Disclosure Statements
*Attachment required if any owner/controller has:*

| Disclosure Item | Citation |
|-----------------|----------|
| Found to have violated PPE Act or similar state law | 5 CCR §71130(c)(1) |
| Been denied any license per B&P Code §480 | 5 CCR §71130(c)(2) |
| Been responsible for closure with unpaid liabilities | 5 CCR §71130(c)(3) |
| Entered stipulated judgment/consent decree on above | 5 CCR §71130(c)(4) |
| Been convicted of misdemeanor or felony per B&P §480(a)(1) | 5 CCR §71130(c)(5) |

#### 3.3 Pending Legal Actions
Explain any pending legal action against institution or ownership involving:
- Fraud or dishonesty
- Financial mismanagement
- Unpaid liabilities
- Student pecuniary loss

| 20-23 | Distance education disclosures | 5 CCR §71716 (if applicable) |

**12 Itemized Charge Categories (5 CCR §71800(e)):**
1. Tuition
2. Registration fee (non-refundable)
3. Equipment
4. Lab supplies/kits
5. Textbooks/learning media
6. Uniforms/protective clothing
7. In-resident housing
8. Tutoring
9. Assessment fees for transfer credit
10. Fees to transfer credits
11. STRF fee (non-refundable)
12. Other charges / Non-institutional charges

**Signature Page Requirements (UNDERLINED CAPITAL LETTERS):**
- Total charges for current enrollment period
- Estimated total charges for entire program
- Total charges student is obligated to pay upon enrollment

### Section 15: Financial Resources and Statements (5 CCR §71240)

| Element | Required |
|---------|----------|
| Confirmation of financial resources per 5 CCR §71745 | Yes |
| **Attachment: Current Audited Financial Statements** | Yes |

**Financial Statement Requirements (5 CCR §74115):**
- Prepared by **independent CPA** (not employee, officer, or board member)
- **GAAP compliant**
- **No more than 120 days old** at submission
- Includes balance sheet and income statement
- Shows sufficient assets for all programs
- Demonstrates ability to pay liabilities and refunds
- Shows capacity for student completion

### Section 20: Copy of Catalog (5 CCR §71290)

#### Catalog Must Contain 42+ Elements (CEC §94909 and 5 CCR §71810)

**Institutional Information:**
1. Institution name
2. Address(es)
3. Telephone number(s)
4. Website address
5. Approval status and means of approval
6. Accreditation status (or statement if unaccredited)
7. BPPE contact information for complaints

**Educational Information:**
8. Mission statement
9. Program objectives
10. Program descriptions
11. Course descriptions
12. Credit hour/clock hour requirements
13. Admissions requirements
14. Transfer of credit policies
15. Transferability disclosure (verbatim language)
16. Unaccredited degree notice (if applicable)

**Faculty and Facilities:**
17. Faculty names and qualifications
18. Facilities description
19. Equipment description
20. Library resources description

**Policies:**
21. Attendance policy
22. Leave of absence policy
23. Grading policy
24. Satisfactory academic progress standards
25. Probation policy
26. Dismissal policy
27. Re-admission policy
28. Grievance/complaint procedures
29. Non-discrimination policy

**Financial Information:**
30. Schedule of total charges
31. Estimated total charges for entire program
32. Payment terms
33. Cancellation and refund policies
34. Refund calculation method
35. STRF disclosure (verbatim language per 5 CCR §76215)
36. Financial aid availability (if applicable)
37. Loan information (if applicable)

**Student Services:**
38. Student services description
39. Housing information (or statement if none)
40. Job placement assistance (if offered)
41. Placement rates (if claims made)

**Other Required Elements:**
42. Catalog effective dates
43. SOC codes for occupational programs
44. License examination passage rates (if licensure program)
45. Completion rates

## Required Attachments Master Checklist

Use this checklist to verify completeness before submission:

### Corporate/Organizational
- [ ] Articles of Incorporation/Organization
- [ ] Bylaws or Operating Agreement
- [ ] Fictitious Business Name Statement (if DBA)
- [ ] Ownership disclosure statements (if applicable)
- [ ] Pending legal action explanations (if applicable)
- [ ] Organization chart
- [ ] Job descriptions for ALL positions
- [ ] CEO, COO, CAO resumes

### Program
- [ ] Mission and objectives statement
- [ ] Program descriptions for each program
- [ ] Curriculum/syllabi for each program
- [ ] 3-year enrollment projections
- [ ] Accreditation plan (if degree programs)
- [ ] Licensing agency approval (if licensure programs)

### Student Documents
- [ ] Enrollment agreement(s)
- [ ] Completed Enrollment Agreement Checklist
- [ ] Instruments of indebtedness (if applicable)
- [ ] Institution catalog
- [ ] Completed Catalog Checklist
- [ ] School Performance Fact Sheet for each program
- [ ] Graduation documents (diploma/certificate) for each program

### Financial
- [ ] Audited financial statements (within 120 days)
- [ ] Financial aid policies (if applicable)

### Personnel
- [ ] Faculty qualifications/credentials for each instructor

### Facility
- [ ] Lease/rental agreements (if applicable)
- [ ] Building diagrams/campus maps
- [ ] Equipment specifications with ownership status
- [ ] Permits/certifications/inspections

### Operations
- [ ] Library and learning resources description
- [ ] Job placement assistance description (if offered)
- [ ] Recordkeeping procedures
- [ ] Self-monitoring procedures
- [ ] All advertising materials

### Administrative
- [ ] Agent for Service of Process designation
- [ ] Signed declarations under penalty of perjury
